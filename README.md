
Bienvenido a mi repositorio de desarrollo. En este espacio comparto proyectos en los que he participado, enfocados en la programación de controladores, automatización y soluciones de hardware/software.

---

Cada proyecto está documentado individualmente. Para revisar el código fuente o los diagramas:

1. **Selecciona el proyecto:** Haz clic en el nombre del firmware o controlador que deseas revisar en el listado de abajo.
2. **Revisa la documentación:** Cada sección incluye una breve introducción con los componentes utilizados y la lógica del sistema.
3. **Descarga los archivos:** 
   * Haz clic en los **tres puntos (`...`)** junto al archivo del proyecto.
   * Selecciona la opción **Download (Descargar)** para obtener los archivos de código o simulación locales.

---


## Tecnologías y Entornos de Desarrollo

* **Microcontroladores:** ESP32, ESP8266, Arduino (Atmega), PIC, STM32.
* **Lenguajes y Herramientas:** C / C++, MicroPython, Lenguaje en Bloques / Ladder.
* **Entornos de Desarrollo (IDEs):** Arduino IDE, STM32CubeIDE, MPLAB X, VS Code (PlatformIO).
* **Simulación y Hardware:** Proteus, Tinkercad, Fritzing, Altium Designer.
